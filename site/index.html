I am Batman FAQ React App

This is an accordion / FAQ solution made with React and Sass. The main component tag can accept a boolean attribute called "allowMultipleOpen" which allows multiple accordion panes to be open simultaneously or only allow one to be open at the time (when the boolean attribute is absent). The questions are also generated from a JSON string so the content can be changed easily and the solution is flexible enough so new themes can be added by defining the primary, secondary, and tertiary color and animations and the collapsed/uncollapsed icons within a SASS switch statement. Hint: Try the Adam West theme.
